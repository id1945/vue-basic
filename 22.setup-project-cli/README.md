## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 22.setup-project-cli
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/22.setup-project-cli
npm install
npm run serve
```
