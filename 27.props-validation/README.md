## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 27.props-validation
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/27.props-validation
npm install
npm run serve
```
