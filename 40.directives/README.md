## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 40.directives
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/40.directives
npm install
npm run serve
```
