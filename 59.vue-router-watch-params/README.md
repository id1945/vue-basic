## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 59.vue-router-watch-params
npm i vue-router --save
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/59.vue-router-watch-params
npm install
npm run serve
```
