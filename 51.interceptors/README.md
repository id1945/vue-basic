## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 51.interceptors
npm install vue-resource --save
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/51.interceptors
npm install
npm run serve
```
