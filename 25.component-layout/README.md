## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 25.component-layout
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/25.component-layout
npm install
npm run serve
```
