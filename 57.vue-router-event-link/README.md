## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 57.vue-router-event-link
npm i vue-router --save
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/57.vue-router-event-link
npm install
npm run serve
```
