## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 60.vue-router-children
npm i vue-router --save
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/60.vue-router-children
npm install
npm run serve
```
