## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 47.transition-animation
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/47.transition-animation
npm install
npm run serve
```
