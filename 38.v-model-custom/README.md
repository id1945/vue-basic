## [1] Create with CLI - Command line
```
npm install -g @vue/cli
vue create 38.v-model-custom
npm run serve
```

## [2] Clone app
```
git clone https://github.com/id1945/vue-basic.git
cd vue-basic/38.v-model-custom
npm install
npm run serve
```
